# 🎓 Sistema Acadêmico UNIFISA

Projeto desenvolvido em **Java** para consolidar os fundamentos de Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades
- **Gestão de Alunos:** Diferenciação entre Graduação (média 7.0) e Pós-Graduação (média 6.0).
- **Gestão de Professores:** Cadastro de professores Efetivos e Temporários com regras de aumento salarial.
- **Disciplinas:** Associação entre um professor e uma lista de alunos (sem duplicidade).

## 🛠️ Conceitos de POO Aplicados
- **Abstração:** Uso de classes e métodos abstratos para padronizar o sistema.
- **Herança:** Reuso de código entre classes mãe e filhas.
- **Encapsulamento:** Proteção de atributos com modificadores `private` e `protected`.
- **Polimorfismo:** Métodos que se comportam de forma diferente para cada tipo de aluno.
